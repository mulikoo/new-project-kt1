# Git Commands Cheatsheet

## Просмотр истории коммитов
```bash
git log --oneline
```

## Перенос изменений между ветками
```bash
git cherry-pick <commit-hash>
```

## Просмотр истории тегов
```bash
git tag -l
```

## Удаление ветки
```bash
git branch -d <branch-name>
```

## Список всех веток
```bash
git branch -a
```
